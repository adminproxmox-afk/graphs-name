Самостійні роботи

Завдання для самостійного вивчення та закріплення матеріалу. Ідеально для підготовки до практичних занять.

1 самостійна робота
СР №1

Самостійна робота до Практичної роботи 1

Самостійна робота для закріплення знань з теми "Графи та способи їх задання". Містить теоретичні питання та практичні завдання.

Цілі роботи:

  • Закріпити знання про основні поняття теорії графів
  • Навчитися будувати графи за заданими умовами
  • Оволодіти різними способами представлення графів
  • Навчитися обчислювати степені вершин

Завдання:

  • Завдання 1: Побудувати граф за заданою матрицею суміжності
  • Завдання 2: Скласти список суміжності для заданого графа
  • Завдання 3: Обчислити степені вершин графа
  • Завдання 4: Визначити, чи є граф повним, регулярним, деревом
  • Завдання 5: Побудувати матрицю інцидентності

Інструкції:

Виконайте всі завдання у зошиті або за допомогою графічного редактора. Для кожного завдання наведіть детальні розрахунки та пояснення.
